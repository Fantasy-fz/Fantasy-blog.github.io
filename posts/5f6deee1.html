里程牌达成，10000次浏览

时间过得真快，
一晃就来到了2022年的最后一天。
很遗憾，今年没有什么拿得出手的成绩。
如果非要说点什么的话，我倒有一份独享的里程牌式成就。
现在很骄傲地与你一起分享。

1. 10000浏览

这个里程碑式成就，就是：
自Fantasy的技术小站建站以来，博客文章的浏览量已经超过了10000次浏览。

这个数据算多吗？

在中国这种人口量级的国家，任何一个热点新闻，又或者抖音爆款视频，浏览器都是1亿起步的。

我这还不到1/10000，何况前后我还用了半年时间，这个数据可以说是“无足轻重”。

但就我个人来说，我很满意，这是一份很不错的成绩。

我让更多的人加入了博客阵营。

值得骄傲，值得炫耀。

2. 十年之约

我是在2021年12月份接触博客的，博客初期没有在意过文章内容，只是一味地寻找友情链接进行交换提高所谓的权重和知名度来满足自己的虚荣心。也正是这样误打误撞的点进了“十年之约”这个浪漫的联盟，被那句“即从加入这个活动起，我们的博客十年不关闭，保持更新和活力！”的slogan所吸引。

我高兴的点击了申请，在提交后故事就这样的开始了。

望走出半生，归来仍是少年

3. 捣鼓最新

在2022年的最后一天

我再次捣鼓了所有站点，站点再一次进化，也算是给了2022年一个交代。

  • Fantasy的技术小站，由原来的bolo改成hexo，写一写 Markdown 文件然后直接推送到 github 仓库就会全自动部署，网站就可以将笔记渲染成十分好看的网页和布局，这种自动化部署和精美的页面是一种能让人十分愉快的体验 (当然前期得折腾很多东西啦😏)。

  • 斯莱特林魔法学院，增加了QQ登录和微信登录，同时设置了服务号推送，并且每天60秒读懂世界

  • 爱情小站，为网站添加灯笼

4. 遇到个女孩

今年没什么好事，就是遇到了个女孩子，动了想和她一辈子在一起的念头，她的出现让我放下以前，我很想一直陪着她

5. 结尾

2023！

保持热爱，奔赴下一场山海。

我已经在期待100000次浏览了
