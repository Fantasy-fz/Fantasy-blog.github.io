redis闪退解决
发表于2023-01-02|更新于2023-01-03|Java|Java
redis启动时闪退，解决方法今天学习redis，双击redis-server.exe时出现闪退的情况，于是上网搜查解决方案。 Windows + R 进入 cmd cd 进入到 redis-server.exe 所在文件夹 写入 redis-server.exe redis.windows.conf，回车 这里给你报了一个错，是 redis 的一个小 bug 再打开一个 cmd 进入 写入 redis-cli，再 exit 退出一下 再返回那个窗口写入 redis-server.exe redis.windows.conf ，回车 发现还是不能用 新建个窗口，写入redis-cli回车，输入shutdown，再exit退出一下 再返回那个窗口写入 redis-server.exe redis.windows.conf ，回车 启动成功
